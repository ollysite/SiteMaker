│                    ScraperPark Studio                           │

│  [스크래퍼] [에디터] [캔버스] [AI 어시스턴트]    [설정] [계정]  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐             │

## 🔄 워크플로우

### 1. 스크래핑 → 캔버스 변환

```
[URL 입력] → [스크래핑] → [프리뷰] → [캔버스로 보내기] → [디자인 편집]
```

### 2. AI 어시스턴트 통합

```
사용자: "헤더 색상을 파란색으로 바꿔줘"
    ↓
[스크래퍼 모드] → AI가 HTML/CSS 직접 수정
[캔버스 모드]  → AI가 요소 속성 변경
```

### 3. 프로젝트 타입

| 타입 | 아이콘 | 설명 |
|------|--------|------|
| `scrape` | 🌐 | 스크래핑된 웹사이트 |
| `canvas` | 🎨 | 캔버스 디자인 |
| `hybrid` | 🔀 | 스크래핑 + 캔버스 연동 |
